# DBMK-Easytime
DBMK-Easytime is een geavanceerd en modern tijd- en weersysteem voor FiveM (ESX Legacy / Lua 5.4), ontworpen om volledige controle te geven over de game-omgeving binnen jouw GTA Roleplay server. Met een strak DBMK-design en krachtige functionaliteiten zorg je voor een realistische en dynamische speelervaring voor alle spelers.
